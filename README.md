# Caixa Eletrônico

Projeto desenvolvido em Java simulando as operações básicas de um caixa eletrônico.

## Funcionalidades

- Depósito de valores
- Saque com verificação de saldo
- Consulta de saldo
- Extrato de transações

## Como usar

Ao iniciar o programa, o usuário deve realizar um depósito inicial para começar a usar o caixa. Em seguida, um menu é exibido com as opções disponíveis, basta digitar o número da opção desejada.

## Tecnologias

- Java
- Orientação a Objetos
